二次修訂摘要

項目修訂內容
① η耦合η* = η_pl·Cγ；Cγ 甘油標定後固定
② K₁/K₂敏感度分析；比值= θ_eff（驗證指標）
③ ε(t)ε_c（碰撞）+ε_s（滑移）+ε_p（顆粒）
④ C_H0.85≤C_H≤1.15；坍落度關聯模型
⑤ 重現性n≥5；CV<10%；ICC≥0.75
⑥ 適用範圍坍落度 50–250 mm，MAS≤19 mm
⑦ 多模型Bingham vs H-B，RMSE/AIC比較
⑧ 無因次化Π_T、Π_τ、Π_η 三個 Pi 群

核心扭矩方程（§2.5）

T(t) = K₁(t)·τ₀,app + K₂(t)·η*·ω(t) + ε(t)
K₁(t) = (1+k_geo)·H(t)·Ar
K₂(t) = K₁(t) / θ_eff
K₁/K₂ = θ_eff ← 驗證指標（偏差<5%）

料面高度 H(t)（§2.3）

H(t) = C_H · θ_eff·H₀ / (θ_eff + Θ(t))
Θ(t) = ∫₀ᵗ ω(s)ds ← 旋轉編碼器積分

臨界停止條件：H(t) ≤ H_yield = τ₀,app / (ρg)

ε(t) 誤差三分量（§2.7）

ε(t) = ε_c(t) + ε_s(t) + ε_p(t)
分量物理意義典型量級
ε_c骨材碰撞衝擊≤15% T_peak
ε_s壁滑移效應≤8% T
ε_p顆粒遷移非均質≤10% T

Tikhonov 反解（§4.3）

x = (AᵀA + λI)⁻¹ · Aᵀb
x = [τ₀,app, η*]ᵀ
  • 準穩態段：|α(t)| < 0.1 rad/s²
  • κ(A) > 100 → 標記低可信度
  • RMSE/T_peak < 5% 為合格

無因次 Pi 群（§8.2）

Π_T = T / (ρgH₀²·Ar·Rout)
Π_τ = τ₀,app / (ρgH₀)
Π_η = η*·ω₀ / (ρgH₀)

跨裝置比較；施工性指標關聯（R²≥0.85）。
